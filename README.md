# Package Butler

See yours version of Node, NPM and Yarn.
See too all global packages installed on yours machine.

## Installation

Clone the project and install the dependencies:

```sh
git clone https://github.com/claudio1091/package-butler.git
cd package-butler
npm install
```

## To run

```sh
npm start
```
